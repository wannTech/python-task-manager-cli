# Python Task Manager CLI

Simple command line task manager built with Python.

## Features

- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Tasks saved to file

## Tech Stack

- Python
- CLI
- File handling
- List & Dictionary

## Example

=== TASK MANAGER ===
1. Add Task
2. View Task
3. Mark Complete
4. Delete Task
5. Exit
